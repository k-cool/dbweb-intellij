

1. 콘솔에 한글 인코딩 보이도록 설정 
tomcat > Edit configurations > server > VM Options 아래 코드 입력 및 서버 재시작

-Dfile.encoding=UTF-8 -Dconsole.encoding=UTF-8


2. 의존성 주입
- POM.xml

        <dependency>
            <groupId>com.servlets</groupId>
            <artifactId>cos</artifactId>
            <version>05Nov2002</version>
        </dependency>

        <dependency>
            <groupId>org.projectlombok</groupId>
            <artifactId>lombok</artifactId>
            <version>1.18.44</version>
            <scope>provided</scope>
        </dependency>


- build.gradle

    // Lombok
    compileOnly 'org.projectlombok:lombok:1.18.44'
    annotationProcessor 'org.projectlombok:lombok:1.18.44'
    testCompileOnly 'org.projectlombok:lombok:1.18.44'
    testAnnotationProcessor 'org.projectlombok:lombok:1.18.44'

    // jstl
    implementation 'javax.servlet:jstl:1.2'

    // cos(mutipart 사용)
    implementation 'com.servlets:cos:05Nov2002'

    // ojdbc -> 디폴트 라이브러리 설정했음 안해도됨
   implementation 'com.oracle.database.jdbc:ojdbc8:19.22.0.0'

  
   // p6spy -> query log
   implementation 'p6spy:p6spy:3.9.1'

  // 로그 적용
   -   src/main/resources/spy.properties 파일 생성하고 아래 내용 넣기

	appender=com.p6spy.engine.spy.appender.StdoutLogger  

   - DBManager에 드라이버 명시해주고 URL 변경
        
        String driver = "com.p6spy.engine.spy.P6SpyDriver";
        String url = "jdbc:p6spy:oracle:thin:@10.1.82.127:1521:XE";

        Class.forName(driver);

3. welcome파일 세팅

- WEB-INF/web.xml -> <web-app> 안에 넣어주기

 <welcome-file-list>
        <welcome-file>index.html</welcome-file>
    </welcome-file-list>
